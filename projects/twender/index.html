About Twender

what is twender?

twender is a small interactive web application that uses machine learning techniques to classify twitter users based on the text of their tweets.

why?

twender was built as a senior project with the goal of learning more about the wonders of machine learning and natural language processing.

well, how good is it?

twender uses tweets that have been labeled based on the first name of the user to simplify the process of collecting training data. that is to say the labels are not always correct. currently twender has 63% accuracy.
